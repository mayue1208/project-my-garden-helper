# 更新日志

## v1.1.0 - 智能浇水提醒系统

### 新增功能
- 实现智能浇水提醒逻辑，根据实际浇水行为动态调整提醒间隔
- 添加3个月重置机制，防止长期偏离初始设置
- 仅对浇水操作应用智能调整逻辑，施肥、修剪等保持固定间隔

### 变更内容
- 修改`cloudfunctions/care/index.js`中的`record`操作，实现智能调整算法
- 修改`cloudfunctions/plant/index.js`中的`create`和`update`操作，添加新字段支持
- 新增`cloudfunctions/data_migrate/index.js`用于数据迁移
- 新增数据库字段：`initialIntervalDays`、`actualIntervalDays`、`lastResetDate`

### 算法说明
1. **提前浇水**：下次提醒间隔 = 实际间隔天数
2. **延迟浇水**：下次提醒间隔 = 原间隔 + 延迟天数
3. **重置机制**：每90天重置回初始设置的间隔天数
4. **边界控制**：间隔限制在3-30天范围内

### 部署要求
- 首次部署需要运行数据迁移脚本为现有浇水配置添加新字段
- 确保云函数正确部署到微信云开发平台

### 影响范围
- 植物浇水提醒功能
- 植物添加/编辑功能
- 养护记录功能
